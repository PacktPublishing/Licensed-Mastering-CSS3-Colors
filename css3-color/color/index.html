Colors start, You create

Colors have been around the web since the very early days. Before the birth of CSS color attributes were baked into the HTML or into images. As the web evolved and moved into a CSS world colors where extracted and moved into style files. Colors have changed the way they are defined and used over the years. The CSS specification has been updated few times to reflect these changes. This chapter is all about colors. Its intention is to walk through colors and understand them in depth. From a theoretical perspective of how colors work through their implementation in a CSS3 environment.

In June, 2011 the W3 incorporated the recommendation of the CSS Color Module Level 3. In it, the group defined what colors are and grouped together the separate specifications that where introduced in HTML 4.01, CSS2, SVG 1, User Interface for CSS3 specifications and other additions into one document. This document showcases all color capabilities of the web with its some new additions and changes. Not all of the specifications will be incorporated by all user agents – instead, agents can develop according to their native capabilities (how many colors they can implement is based on the device itself and display capabilities).

Understanding RGB

From the very early days of the web colors where defined with RGB values – Red, Green, Blue. The mix of these three base colors is what creates the millions of colors available on most modern devices. Each color channel has 255 color value options combined into a total of 16,581,375 color combination possibilities. In this lecture we will meet the common ways to describe color in on digital devices using these color profile options. By the way, You might of heard of 16Bit displays or 32Bit – these differences don't really mean much when it comes to digital colors as most modern interfaces only are capable of showing a variety of 16 million colors – monitors with 4 color profiles instead of 3 create a vastly larger color options – but the web doesn't really support more color profiles currently. In the olden days of the web when monitors where less robust then they are today there was a term called “Web Safe” that was a subset of 255 colors in total. Now days, with most digital display system supporting millions of colors this term is obsolete. By the end of this lecture, you will know how to create color profiles in RGB.

CSS2 System Colors (deprecated)

In our earlier lectures we talked about colors and web-safe colors. There are some colors that are reserved for the usage of the OS/browser. Through the usage of CSS2 System colors - we can ask the browser for the system colors. With this color set we can incorporate system colors into our design.

The CSS2 System colors have been deprecated in favor of CSS3 UI specification (still in development as written). Access to the color scheme of the underlying desktop/operating system can create more easily shady interfaces that could confuse users. As its original intent was to enable smoother interface interactions the feature has been removed from the recommendation in favor of using UI elements. With that said, this feature is still widely supported and as long as the CSS3 UI elements won't be fully accepted and finalized, it will be around to be used.
